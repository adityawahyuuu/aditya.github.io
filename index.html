Bookshelf API
Screenshot

Bookshelf merupakan tugas akhir dari course backend pemula dicoding, kemudian saya improvisasi hingga dapat terhubung dengan mongodb. Teknologi yang dipakai yaitu hapi, mongodb, nodejs dan ejs.

    Fitur

  • Menambahkan Buku
  • Melihat Buku
  • Melihat Detail Buku
  • Mengedit Buku
  • Menghapus Buku
  • Authentikasi dengan @hapi/jwt dan jsonwebtoken
  • Login/Logout
  • Satu user satu collection database MongooDB
